🔹 CSS

css

  width: 300px;               /* tuỳ bạn */
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 10px;
}

.gallery img {
  width: 100%;
  aspect-ratio: 1 / 1;        /* ép ảnh thành hình vuông */
  object-fit: cover;
  border-radius: 10px;
}

📌 Kết quả: 2 ảnh trên + 2 ảnh dưới = hình vuông

css

  flex-wrap: wrap;
  gap: 10px;
}

  aspect-ratio: 1 / 1;
  object-fit: cover;
}
